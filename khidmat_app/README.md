# Khidmat App

A mobile app built with Flutter for the Khidmat service booking platform. 
Connects with the FastAPI backend using AI agents to find the best service providers.

## Features
- **Agent Trace Logs**: Complete transparency into the AI workflow.
- **Provider Ranking**: See top AI-recommended providers.
- **Booking Flow**: Complete your booking directly in-app.

## How to Run
1. Ensure the FastAPI backend is running on `http://127.0.0.1:8000`.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to start the app.
